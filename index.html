universe

Cross-platform dotfiles managed with lnk. Opinionated macOS and Linux configurations for development.

Quick Install

curl -fsSL https://universe.nibras.co/install.sh | bash

Features

  • Cross-platform via lnk manifests
  • Fish shell with custom functions
  • Homebrew bundle for macOS
  • Cursor/VS Code settings sync
  • Idempotent installation scripts

Structure

  • darwin.lnk — macOS-specific symlinks
  • linux.lnk — Linux-specific symlinks
  • scripts/ — setup and utility scripts
  • .config/ — application configs

Usage

  • lnk pull — apply dotfiles
  • lnk pull -H darwin — macOS only
  • brew bundle --global — install packages
